# Webhook deployer

Этот проект создавался для запуска в докере из корневой директории проекта,
нужно создать контейнер и примаунтить к нему:

```text
/var/run/docker.sock:/var/run/docker.sock
./:/repo
~/.ssh/:/root/.ssh/
```
.env будет скопирован в образ из докерфайла
